# Casino Website Layout — Demo Template

A **portfolio demo layout** for showing casinos when pitching website updates. Same structure and methods as modern casino sites (hero, nav, play/packages/entertainment, stay, gallery, infrastructure). **All content and images are placeholders** — fake copy, generic stock photos, and placeholder links only.

## Use case

- Show this to casino clients as a **layout demo**
- “We can build or refresh your site with this structure”
- Replace branding, copy, and images with the client’s real content when they sign on

## What’s included (all placeholder)

- **Hero** — Full-height section with video/image background and sample headline
- **Nav** — Play (Live / Online), Packages (INR / USD), Entertainment, Rewards
- **Quick links** — Card grid for main sections
- **Live Casino** — Sample copy for floor and VIP
- **Nightlife & City** — Placeholder for local experiences/packages
- **Luxury Stay** — Placeholder for accommodation
- **Gallery** — Generic stock image grid (replace with client photos)
- **Infrastructure & Security** — Example capabilities you can implement (for your pitch)
- **Footer** — Placeholder links and “© [Your Casino Name]”

## Placeholder content

- **Branding:** “Luxe Casino” (replace with client name)
- **Copy:** Clearly marked as sample/placeholder
- **Images:** Unsplash/Coverr generic (casino/luxury) — not real venue photos
- **Links:** `#` or in-page anchors only; no real social or booking URLs
- **Badge:** “Demo layout — placeholder content only” at top so it’s obvious in meetings

## Run locally

```bash
cd ballys-colombo-website
npx serve .
```

Or open `index.html` in a browser.

## Customization for a real client

1. Replace “Luxe Casino” and logo with client name.
2. Swap all placeholder copy with their messaging.
3. Replace hero image/video and gallery images with their assets.
4. Set real links (social, packages, booking, contact).
5. Remove or keep the “Demo layout” badge (e.g. remove for production).
6. Update footer copyright to client name.
